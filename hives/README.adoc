= Hives Directory
:toc:

== Overview

This directory contains git submodules for each My-Lang dialect hive.

== Hive Layout Contract

Each hive must conform to the layout contract defined in `.machine_read/SPEC.playground.scm`:

. `README.md` or `README.adoc` - Hive documentation
. `demo/` directory - Runnable examples
. `justfile` or `Mustfile` - With 'demo' target
. `.hive-meta.scm` - Hive metadata

== Adding a Hive

[source,bash]
----
git submodule add https://github.com/hyperpolymath/{dialect}-hive hives/{dialect}-hive
git commit -m "Add {dialect}-hive submodule"
----

IMPORTANT: Always pin submodule refs to specific commits. No floating HEAD refs.
